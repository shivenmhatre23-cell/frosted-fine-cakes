# 🍰 Frosted & Fine Cakes

A modern, responsive business website for **Frosted & Fine Cakes**, a homemade cake brand offering beautifully crafted cakes for birthdays, anniversaries, festivals, and special occasions.

This project was built as a complete frontend development project using **HTML5, CSS3, and JavaScript**, following a professional development workflow from planning and design to testing, deployment, and production release.
